## Common Utility Methods

### String Class
  - it is a reference type
  - strings are immutable (we cant change the string)
  - any methods that changes a string returns a new string

|Method|Purpose|Example|
|------|-------|-------|
|`length()`|Returns string length	|`"Hello".length()` → `5`|
|`isEmpty()`|Checks if string is empty|`"hello".isEmpty()` → `false`|
|`equals(str)`|Compares string values|`"Hello".equals("Hello")`|
|`indexOf(str)`|Returns starting index of substring|`"Hello".indexOf("He")` → `0`|
|`indexOf(val)`|Returns index of first occurrence|`"Hello".indexOf("o")` → `4`|

|`substring(start,end)`|Extracts part of string	|`"Hello".substring(1,4)` → `"ell"`|

### Constants

- to declare a constant we user final keyword


### Casting

- Implicit Casting (no data loss): byte > Short > int > long > float > double
```java
short x = 1;
int y = x + 1; // implicit casting
System.out.println(y);
```

- Explicit Casting (data loss): double > float > long > int > char > short > byte
```java
double x = 1.1;
int y = (int)x + 2; // Explicit casting
System.out.println(y); // 3
```

## Java Arrays

- **method overloading:** multiple methods with the same name but different parameter (Ex: Arrays.toString())
- **method overridding:** the method of the subclass class overrides the method of the superclass

```java
Single Dimension Array:

    int[] numbers = new int[5]; // declaration - uninitialized values: 0
    int[] numbers = new int[]{3,4}; // declaration and initialization (Shouldn't declare the size to initialize)
    int[] numbers = {2,3,4,1,5};
    System.out.println(Arrays.toString(numbers)); // [2, 3, 4, 1, 5]
    Arrays.sort(numbers);
    System.out.println(Arrays.toString(numbers)); // [1, 2, 3, 4, 5]

Multi Dimension Array:

    int[][] numbers = new int[][]{{1,2,3},{0,0,0}};
    System.out.println(Arrays.deepToString(numbers)); // [[1, 2, 3], [0, 0, 0]]
    int[][] multiArray = {{1,2},{0,0,0}};
    System.out.println(Arrays.deepToString(multiArray)); // [[1, 2], [0, 0, 0]]
    int[][] multiArray2 = new int[2][3];
    System.out.println(Arrays.deepToString(multiArray2)); // [[0, 0, 0], [0, 0, 0]]


```

- **Static return**

```java
return new int[] {1,2,3};
```

- **Methods**

| Method | Description | Example |
|---|---|---|
| `length` | Returns size of array | `arr.length` |
| `sort(arr)` | Sorts array in ascending order | `Arrays.sort(arr)` |
| `equals(arr1,arr2)` | Checks two arrays are equal | `Arrays.equals(arr1,arr2)` |
| `toString(arr)` | Converts array to string | `Arrays.toString(arr)` |
| `deepToString(arr)` | Converts multi-dimension array to string | `Arrays.deepToString(numbers)` |

## Set

### HashSet

- A HashSet is a data structure that stores **unique elements only**
- Does **not maintain order** (no insertion order, no sorting)
- Internally uses a **HashMap**
- Not synchronized, not thread-safe

- **Declaration**

```java
HashSet<Integer> set = new HashSet<>();
HashSet<String> set2 = new HashSet<>(Arrays.asList("a", "b", "c"));
```

- **Methods**

| Method | Description | Example |
|---|---|---|
| `add(val)` | Adds element (returns false if already exists) | `set.add(10);` |
| `remove(val)` | Removes element if present | `set.remove(10);` |
| `contains(val)` | Checks if element exists | `set.contains(5);` |
| `size()` | Returns number of elements | `set.size();` |
| `clear()` | Removes all elements | `set.clear();` |
| `isEmpty()` | Checks if set is empty | `set.isEmpty();` |
| `iterator()` | Returns iterator (no guaranteed order) | `set.iterator();` |
| `for-each` | Iterate through elements | `for(int x : set) {}` |
| `addAll(collection)` | Adds all elements from collection | `set.addAll(list);` |
| `removeAll(collection)` | Removes all elements from collection | `set.removeAll(list);` |
| `retainAll(collection)` | Keeps only common elements | `set.retainAll(list);` |
| `toArray()` | Converts set to array | `set.toArray();` |

### TreeSet

- A TreeSet is a sorted set that stores unique elements in ascending order (default)
- Internally uses a **Red-Black Tree**
- Not synchronized, not thread-safe

- **Declaration**

```java
TreeSet<Integer> ts = new TreeSet<>(); // default ascending order

TreeSet<Integer> desc = new TreeSet<>((a, b) -> b - a); // descending order

TreeSet<Integer> ts2 = new TreeSet<>(Arrays.asList(3, 1, 2));
```

- **Methods**

| Method | Description | Example |
|---|---|---|
| `add(val)` | Adds element (no duplicates allowed) | `ts.add(10);` |
| `remove(val)` | Removes element | `ts.remove(10);` |
| `contains(val)` | Checks if element exists | `ts.contains(5);` |
| `first()` | Returns smallest element | `ts.first();` |
| `last()` | Returns largest element | `ts.last();` |
| `higher(val)` | Returns next greater element | `ts.higher(10);` |
| `lower(val)` | Returns next smaller element | `ts.lower(10);` |
| `ceiling(val)` | Returns smallest element ≥ val | `ts.ceiling(10);` |
| `floor(val)` | Returns largest element ≤ val | `ts.floor(10);` |
| `pollFirst()` | Removes & returns smallest element | `ts.pollFirst();` |
| `pollLast()` | Removes & returns largest element | `ts.pollLast();` |
| `size()` | Returns number of elements | `ts.size();` |
| `clear()` | Removes all elements | `ts.clear();` |
| `isEmpty()` | Checks if empty | `ts.isEmpty();` |
| `iterator()` | Returns iterator (sorted order) | `ts.iterator();` |

- **Important Points**

- Elements are always **sorted**
- No duplicates allowed
- All operations → **O(log n)**
- Used when you need **sorted + unique elements**

## Queue

### PriorityQueue

- A PriorityQueue is a heap-based data structure where elements are ordered based on priority (default: min-heap)
- Not synchronized, not thread-safe

- **Declaration**

```java
PriorityQueue<Integer> pq = new PriorityQueue<>(); // min-heap (default)
PriorityQueue<Integer> maxHeap = new PriorityQueue<>((a, b) -> b - a); // max-heap
PriorityQueue<int[]> pqArr = new PriorityQueue<>((a, b) -> a[0] - b[0]); // based on first element
PriorityQueue<Integer> pq2 = new PriorityQueue<>(Arrays.asList(1, 3, 2));
```

- **Methods**

| Method | Description | Example |
|---|---|---|
| `offer(val)` | Inserts element into queue (preferred) | `pq.offer(10);` |
| `add(val)` | Inserts element (throws exception if fails) | `pq.add(20);` |
| `peek()` | Returns top element without removing | `pq.peek();` |
| `poll()` | Removes and returns top element | `pq.poll();` |
| `remove()` | Removes top element (throws exception if empty) | `pq.remove();` |
| `remove(val)` | Removes specific element | `pq.remove(10);` |
| `contains(val)` | Checks if element exists | `pq.contains(5);` |
| `size()` | Returns number of elements | `pq.size();` |
| `clear()` | Removes all elements | `pq.clear();` |
| `isEmpty()` | Checks if queue is empty | `pq.isEmpty();` |
| `iterator()` | Returns iterator (not sorted order) | `pq.iterator();` |
| `toArray()` | Converts queue to array | `pq.toArray();` |

### Deque

- A Deque (Double Ended Queue) allows insertion and deletion from both ends
- Can be used as **queue (FIFO)** or **stack (LIFO)**
- Common implementations:
  - `ArrayDeque` (fast, preferred)
  - `LinkedList`
- Not synchronized, not thread-safe

- **Declaration**

```java
Deque<Integer> dq = new ArrayDeque<>();

Deque<Integer> dq2 = new LinkedList<>();

Deque<Integer> dq3 = new ArrayDeque<>(Arrays.asList(1, 2, 3));
```
- **Methods**

| Method | Description | Example |
|---|---|---|
| `addFirst(val)` | Adds element at front | `dq.addFirst(10);` |
| `addLast(val)` | Adds element at rear | `dq.addLast(20);` |
| `offerFirst(val)` | Adds at front (returns false if fails) | `dq.offerFirst(5);` |
| `offerLast(val)` | Adds at rear (returns false if fails) | `dq.offerLast(15);` |
| `removeFirst()` | Removes & returns front element | `dq.removeFirst();` |
| `removeLast()` | Removes & returns last element | `dq.removeLast();` |
| `pollFirst()` | Removes & returns front (null if empty) | `dq.pollFirst();` |
| `pollLast()` | Removes & returns last (null if empty) | `dq.pollLast();` |
| `getFirst()` | Returns front element | `dq.getFirst();` |
| `getLast()` | Returns last element | `dq.getLast();` |
| `peekFirst()` | Returns front (null if empty) | `dq.peekFirst();` |
| `peekLast()` | Returns last (null if empty) | `dq.peekLast();` |
| `push(val)` | Pushes element (stack behavior) | `dq.push(10);` |
| `pop()` | Removes top element (stack behavior) | `dq.pop();` |
| `size()` | Returns number of elements | `dq.size();` |
| `clear()` | Removes all elements | `dq.clear();` |
| `isEmpty()` | Checks if empty | `dq.isEmpty();` |
| `iterator()` | Returns iterator (front to rear) | `dq.iterator();` |
