# Java Collections Big-O Cheat Sheet

Below are the Big-O performance of common functions of different Java Collections.

<summary>Table of contents</summary>

- [Set](#set)

### List

| List                  | Add  | Remove | Get  | Contains | Next | Data Structure |
|-----------------------|------|--------|------|----------|------|----------------|
| ArrayList             | O(1) | O(n)   | O(1) | O(n)     | O(1) | Array          |
| LinkedList            | O(1) | O(1)   | O(n) | O(n)     | O(1) | Linked List    |
| CopyOnWriteArrayList  | O(n) | O(n)   | O(1) | O(n)     | O(1) | Array          |

---

### Set

| Set                    | Add      | Remove   | Contains | Next | Size | Data Structure                |
|------------------------|----------|----------|----------|------|------|-------------------------------|
| HashSet                | O(1)     | O(1)     | O(1)     | O(1) | O(1) | Hash Table                    |
| LinkedHashSet          | O(1)     | O(1)     | O(1)     | O(1) | O(1) | Hash Table + Linked List      |
| EnumSet                | O(1)     | O(1)     | O(1)     | O(1) | O(1) | Bit Vector                    |
| TreeSet                | O(log n) | O(log n) | O(log n) | O(1) | O(1) | Red-Black Tree                |
| CopyOnWriteArraySet    | O(n)     | O(n)     | O(n)     | O(1) | O(1) | Array                         |
| ConcurrentSkipListSet  | O(log n) | O(log n) | O(log n) | O(1) | O(1) | Skip List                     |

---

### Queue

| Queue                   | Offer    | Peek | Poll     | Remove | Size | Data Structure   |
|--------------------------|----------|------|----------|--------|------|------------------|
| PriorityQueue            | O(log n) | O(1) | O(log n) | O(n)   | O(1) | Binary Heap      |
| LinkedList               | O(1)     | O(1) | O(1)     | O(1)   | O(1) | Linked List      |
| ArrayDeque               | O(1)     | O(1) | O(1)     | O(n)   | O(1) | Array            |
| ConcurrentLinkedQueue    | O(1)     | O(1) | O(1)     | O(n)   | O(1) | Linked List      |
| ArrayBlockingQueue       | O(1)     | O(1) | O(1)     | O(n)   | O(1) | Array            |
| PriorityBlockingQueue    | O(log n) | O(1) | O(log n) | O(n)   | O(1) | Binary Heap      |
| SynchronousQueue         | O(1)     | O(1) | O(1)     | O(1)   | O(1) | None             |
| DelayQueue               | O(log n) | O(1) | O(log n) | O(n)   | O(1) | Binary Heap      |
| LinkedBlockingQueue      | O(1)     | O(1) | O(1)     | O(n)   | O(1) | Linked List      |

---

### Map

| Map                    | Get      | ContainsKey | Next | Data Structure           |
|------------------------|----------|-------------|------|--------------------------|
| HashMap                | O(1)     | O(1)        | O(1) | Hash Table               |
| LinkedHashMap          | O(1)     | O(1)        | O(1) | Hash Table + Linked List |
| IdentityHashMap        | O(1)     | O(1)        | O(1) | Array                    |
| WeakHashMap            | O(1)     | O(1)        | O(1) | Hash Table               |
| EnumMap                | O(1)     | O(1)        | O(1) | Array                    |
| TreeMap                | O(log n) | O(log n)    | O(1) | Red-Black Tree           |
| ConcurrentHashMap      | O(1)     | O(1)        | O(1) | Hash Table               |
| ConcurrentSkipListMap  | O(log n) | O(log n)    | O(1) | Skip List                |
