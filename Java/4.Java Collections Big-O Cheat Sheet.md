### List

| List                  | Add  | Remove | Get  | Contains | Next | Data Structure |
|-----------------------|------|--------|------|----------|------|----------------|
| ArrayList             | O(1)*| O(n)   | O(1) | O(n)     | O(1) | Dynamic Array |
| LinkedList            | O(1) | O(1)   | O(n) | O(n)     | O(1) | Doubly Linked List |
| CopyOnWriteArrayList  | O(n) | O(n)   | O(1) | O(n)     | O(1) | Array (copy on write) |

> *O(1) amortized (resize → O(n))

### Map

| Map                    | Put      | Get      | Remove   | ContainsKey | Next | Data Structure |
|------------------------|----------|----------|----------|-------------|------|----------------|
| HashMap                | O(1)*    | O(1)*    | O(1)*    | O(1)*       | O(1) | Hash Table (tree after collisions) |
| LinkedHashMap          | O(1)     | O(1)     | O(1)     | O(1)        | O(1) | Hash Table + Doubly Linked List |
| IdentityHashMap        | O(1)     | O(1)     | O(1)     | O(1)        | O(1) | Array (reference equality) |
| WeakHashMap            | O(1)     | O(1)     | O(1)     | O(1)        | O(1) | Hash Table (GC-based keys) |
| EnumMap                | O(1)     | O(1)     | O(1)     | O(1)        | O(1) | Array (enum index) |
| TreeMap                | O(log n) | O(log n) | O(log n) | O(log n)    | O(1) | Red-Black Tree |
| ConcurrentHashMap      | O(1)     | O(1)     | O(1)     | O(1)        | O(1) | Hash Table (CAS, no locks) |
| ConcurrentSkipListMap  | O(log n) | O(log n) | O(log n) | O(log n)    | O(1) | Skip List |

> *Worst case → O(n), Java 8+ → O(log n) (tree bins)

## ⚡ Golden Rules (Interview Ready)

- Hash → **Fast (O(1))**
- Tree → **Sorted (O(log n))**
- Linked → **Order maintained**
- Heap → **Priority based**
- CopyOnWrite → **Read heavy only**
- Concurrent → **Thread-safe optimized**

---

## 🚫 Avoid (Legacy)

- Vector ❌ → use `ArrayList`
- Stack ❌ → use `ArrayDeque`
- Hashtable ❌ → use `HashMap`

---

## 🎯 Defaults (Always Safe)

- List → `ArrayList`
- Set → `HashSet`
- Map → `HashMap`
- Queue → `ArrayDeque`
